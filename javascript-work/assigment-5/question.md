Q1. Write a program to take email as an input from the user through a prompt. Validate email address with following evaluation.
a. This string must contain @
b. The string must contact chars
c. The @ must have at least 1 char in front
d. The string must contain .(dot) after @ and at least 1 char.
e. If the email is correct, alert the user that your email is correct.

Q2. Write a program to take password as an input from the user through a prompt. The
password must qualify these requirements:

a. It should contain alphabets and numbers
b. It should not start with a number
c. It must at least 8 characters long
d. If the password does not meet the above requirements, again prompt
the user to enter a valid password. For character codes of a-z,
A-Z & 0-9.
e. If the password is correct, alert the user that your password is correct.
